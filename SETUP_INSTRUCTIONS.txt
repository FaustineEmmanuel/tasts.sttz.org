FASTEST WAY TO GET STARTED (5 minutes):
================================================================================

METHOD 1: XAMPP (Easiest for beginners)
--------------------------------------
1. Download XAMPP from: https://www.apachefriends.org/
2. Install with default settings
3. Extract TASTS-2025 folder to: xampp/htdocs/tasts2025/
4. Open XAMPP Control Panel
5. Click "Start" next to Apache
6. Open browser and visit: http://localhost/tasts2025/
7. DONE! Your website is live.

METHOD 2: Local PHP Server (No installation needed)
---------------------------------------------------
1. Open terminal/command prompt
2. Navigate to TASTS-2025 folder
3. Type: php -S localhost:8000
4. Open browser and visit: http://localhost:8000/
5. DONE! Your website is live.

METHOD 3: Web Hosting (For production)
--------------------------------------
1. Use FTP client (FileZilla - free download)
2. Upload all files to your hosting account
3. Visit your website URL
4. DONE! Your website is live online.

================================================================================
TROUBLESHOOTING:
================================================================================

Problem: "Page not found"
Solution: Make sure all files are in the correct folder structure

Problem: "Form not submitting"
Solution: Check that submit_registration.php is in the root directory

Problem: "Emails not sending"
Solution: 
  - Check SMTP settings in config.php
  - Look in logs/error.log for error messages
  - Enable PHP mail() in php.ini

Problem: "Styling looks broken"
Solution: Clear browser cache (Ctrl+Shift+Delete) and refresh
