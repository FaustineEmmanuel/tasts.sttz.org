Sustainable Tourism Tanzania (STTZ) Website
A professional, responsive website for Sustainable Tourism Tanzania built with PHP, MySQL, CSS, and JavaScript.

Features
Fully Responsive Design - Optimized for mobile, tablet, and desktop devices
Professional Color Scheme - Green and blue tones matching the STTZ logo
Contact Form - Sends emails to info@sttz.org and stores in MySQL database
Smooth Animations - Scroll effects, fade-ins, and hover transitions
SEO Optimized - Meta tags, semantic HTML, and performance optimization
XAMPP Compatible - Ready to run on local XAMPP environment

Installation for XAMPP
Copy files to XAMPP htdocs folder:

Copy all files to: C:\xampp\htdocs\sttz\
Start XAMPP:

Start Apache
Start MySQL
Setup Database:

Visit: http://localhost/sttz/setup-database.php
This will create the database and tables automatically
Access the Website:

Homepage: http://localhost/sttz/
Contact Form: http://localhost/sttz/contact.php
Database Configuration
The website uses MySQL database with the following default settings:

Host: localhost
Username: root
Password: (empty)
Database Name: sttz_website
You can modify these settings in config/database.php

Email Configuration
Contact form submissions are:

Saved to MySQL database (contact_submissions table)
Sent via email to info@sttz.org
To configure email settings, edit config/email.php
