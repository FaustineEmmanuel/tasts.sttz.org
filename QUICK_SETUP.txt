QUICK START (3 EASY STEPS):
==========================

STEP 1: Download & Install XAMPP
   → Go to: https://www.apachefriends.org/
   → Download XAMPP for your system (Windows/Mac/Linux)
   → Run installer and follow prompts
   → Remember installation location!

STEP 2: Copy Website Files
   → Find XAMPP htdocs folder:
      Windows: C:\xampp\htdocs
      Mac:     /Applications/XAMPP/htdocs
      Linux:   /opt/lampp/htdocs
   
   → Create new folder: "tasts2025"
   → Copy ALL website files into this folder
   
   Files to copy:
   - index.php
   - config/ (folder with settings.php, functions.php)
   - pages/ (folder with register.php)
   - includes/ (folder with header.php, footer.php)
   - assets/ (folder with css/ and js/ subfolders)

STEP 3: Start XAMPP & Visit Website
   → Open XAMPP Control Panel
   → Click "Start" next to Apache
   → Open browser and go to: http://localhost/tasts2025/
   → Done! Website is live!

CUSTOMIZE IT:
=============
Change Email:
   → Edit config/settings.php
   → Change RECIPIENT_EMAIL to your email

Change Date/Location:
   → Edit config/settings.php
   → Update EVENT_DATE and EVENT_LOCATION

Add Speakers:
   → Edit config/functions.php
   → Add new speaker in getSpeakers() function

Change Colors:
   → Edit assets/css/style.css
   → Modify :root CSS variables

TROUBLESHOOTING:
================
Website not loading?
   → Check if Apache is running in XAMPP Control Panel
   → URL should be http://localhost/tasts2025/ (exact match)

Form not submitting?
   → Press F12 to open Developer Tools
   → Check Console tab for error messages
   → Check that logs/ folder exists

Colors look different?
   → Clear browser cache (Ctrl+Shift+Delete)
   → Try different browser to test

Emails not sending?
   → Check logs/registrations.log (data is saved there)
   → XAMPP logs emails locally by default
   → For production, configure mail service
